## ‚úÖ PART 1: OPERATORS in JavaScript

**Operators** are symbols that perform operations on values (like math, comparison, assignment, etc.).

---

### üî∏ 1. **Arithmetic Operators** (for numbers)

| Operator | Meaning             | Example  | Result |
| -------- | ------------------- | -------- | ------ |
| `+`      | Addition            | `5 + 2`  | `7`    |
| `-`      | Subtraction         | `5 - 2`  | `3`    |
| `*`      | Multiplication      | `5 * 2`  | `10`   |
| `/`      | Division            | `10 / 2` | `5`    |
| `%`      | Modulus (remainder) | `5 % 2`  | `1`    |
| `**`     | Exponent (power)    | `2 ** 3` | `8`    |

---

### üî∏ 2. **Assignment Operators**

| Operator | Example  | Same as     |
| -------- | -------- | ----------- |
| `=`      | `x = 10` | assign 10   |
| `+=`     | `x += 5` | `x = x + 5` |
| `-=`     | `x -= 3` | `x = x - 3` |
| `*=`     | `x *= 2` | `x = x * 2` |

---

### üî∏ 3. **Comparison Operators** (used in conditions)

| Operator | Meaning          | Example     | Result  |
| -------- | ---------------- | ----------- | ------- |
| `==`     | Equal (loose)    | `5 == '5'`  | `true`  |
| `===`    | Strict equal     | `5 === '5'` | `false` |
| `!=`     | Not equal        | `5 != 3`    | `true`  |
| `!==`    | Strict not equal | `5 !== '5'` | `true`  |
| `>`      | Greater than     | `5 > 3`     | `true`  |
| `<`      | Less than        | `2 < 4`     | `true`  |
| `>=`     | Greater or equal | `4 >= 4`    | `true`  |
| `<=`     | Less or equal    | `3 <= 4`    | `true`  |

---

### üî∏ 4. **Logical Operators**

| Operator | Meaning              | Example         | Result            |        |   |         |      |
| -------- | -------------------- | --------------- | ----------------- | ------ | - | ------- | ---- |
| `&&`     | AND (both true)      | `true && false` | false             |        |   |         |      |
| \`       |                      | \`              | OR (any one true) | \`true |   | false\` | true |
| `!`      | NOT (opposite value) | `!true`         | false             |        |   |         |      |

## ‚úÖ PART 2: CONDITIONS in JavaScript

### What are Conditions?

They **make decisions** based on true/false values using `if`, `else if`, and `else`.

---

### üî∏ 1. **If Statement**

```javascript
let age = 18;
if (age >= 18) {
  console.log("You can vote.");
}
```

---

### üî∏ 2. **If...Else**

```javascript
let age = 16;
if (age >= 18) {
  console.log("You can vote.");
} else {
  console.log("You are too young to vote.");
}
```

---

### üî∏ 3. **If...Else If...Else**

```javascript
let marks = 75;

if (marks >= 90) {
  console.log("Grade A");
} else if (marks >= 75) {
  console.log("Grade B");
} else {
  console.log("Grade C");
}
```

---

### üî∏ 4. **Ternary Operator** (Short version of if...else)

```javascript
let age = 20;
let result = (age >= 18) ? "Adult" : "Minor";
console.log(result);  // Adult
```

---

### üî∏ 5. **Switch Statement** (when checking multiple exact matches)

```javascript
let color = "red";

switch (color) {
  case "red":
    console.log("Stop");
    break;
  case "yellow":
    console.log("Wait");
    break;
  case "green":
    console.log("Go");
    break;
  default:
    console.log("Invalid color");
}
```

---
